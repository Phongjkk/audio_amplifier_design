# Audio Amplifier Design

Audio amplifier circuit design and simulation using **LTspice**.

## Overview
This project focuses on the design and simulation of an audio amplifier circuit
using **BJTs and basic electronic components**.

## Objectives
- Design a stable audio amplifier circuit
- Analyze voltage gain and frequency response
- Verify circuit behavior through LTspice simulation

## Tools
- LTspice

## Circuit Design
The amplifier circuit is designed using BJT transistors and basic components.
The schematic file is provided in the `circuit/` directory.

## Simulation
Simulation results include:
- AC analysis
- Transient response
